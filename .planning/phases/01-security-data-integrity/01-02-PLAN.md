<objective>
Prevent binary File objects from being serialised into localStorage when an order wizard draft is saved.

Purpose: The current useEffect in OrderFormContext.tsx writes the entire form state — including the image_files field (which holds File objects) — directly to localStorage via JSON.stringify. File objects do not serialise; they become empty `{}` objects or throw a QuotaExceededError when images are large. FixFormContext.tsx already has the correct fix. This plan replicates that exact pattern in OrderFormContext.tsx.

Output: OrderFormContext.tsx useEffect strips image_files before serialisation, matching FixFormContext.tsx lines 29-33 exactly.
</objective>
